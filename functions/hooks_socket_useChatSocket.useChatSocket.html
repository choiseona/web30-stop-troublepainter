Function useChatSocket

채팅 소켓 연결과 메시지 처리를 관리하는 커스텀 훅입니다.

Remarks

  • 소켓 연결 생명주기 관리
  • 메시지 송수신 처리
  • 메시지 영속성을 위한 채팅 스토어 통합

Example

const { messages, isConnected, sendMessage } = useChatSocket();

// 메시지 전송
sendMessage("안녕하세요");
